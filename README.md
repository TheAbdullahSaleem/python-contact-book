# 📒 Python Contact Book

A command-line Contact Book built with Python that allows users to store and manage contacts using file-based storage.

Each contact includes a name, phone number, and email address. The application supports creating, viewing, searching, updating, and deleting contacts.

---

## Features

- ➕ Add new contacts
- 📋 View all contacts
- 🔍 Search contacts by name
- ✏️ Update existing contacts
- ❌ Delete contacts
- 💾 Persistent storage using a text file
- ✅ Input validation
- ⚠️ File handling and error handling

---

## Technologies Used

- Python 3

---

## How to Run

Clone the repository:

```bash
git clone https://github.com/TheAbdullahSaleem/python-contact-book.git
```

Run the program:

```bash
python main.py
```
